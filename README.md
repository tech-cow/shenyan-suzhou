# 深言速舟官网 · Shenyan Suzhou website

深言速舟（宁波）智能科技有限公司官方网站（测试版）。

- 单页、自包含静态站点（`index.html`），内联 CSS + JS，无外部依赖。
- 中英双语，右上角一键切换。
- 通过 GitHub Pages 部署，公开访问地址：`https://tech-cow.github.io/shenyan-suzhou/`

## 本地预览

直接用浏览器打开 `index.html` 即可；或起一个本地静态服务器：

```bash
python -m http.server 8000
# 打开 http://localhost:8000
```
